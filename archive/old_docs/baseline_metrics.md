## 四、tstat 显著性参考标准

| |tstat| 范围 | 含义 |
|------------|------|
| > 1.96 | ✅ Alpha 在 95% 置信水平显著 |
| 1.645 ~ 1.96 | ⚠️ Alpha 在 90% 置信水平显著（边缘） |
| 1.5 ~ 1.645 | ⚠️ 边缘，弱显著 |
| < 1.5 | ❌ Alpha 不显著 |

## 五、tstat 代码实现位置

```python
# backtest/portfolio_backtester.py, 第 485-494 行
daily_alpha = daily_returns - bench_daily_ret.values[:len(daily_returns)]
alpha_mean = np.mean(daily_alpha)
alpha_std = np.std(daily_alpha, ddof=1)
# t-stat = mean(alpha) / (std(alpha) / sqrt(N))
alpha_tstat = (alpha_mean / (alpha_std / np.sqrt(len(daily_alpha)))) if alpha_std > 0 else 0
# Information Ratio = annualized alpha / tracking error
tracking_error = alpha_std * np.sqrt(252)
information_ratio = (alpha_mean * 252) / tracking_error if tracking_error > 0 else 0
```

**计算方式**：对每日超额收益（组合日收益 - 基准日收益）做 t 检验，零假设为 Alpha = 0。

## 六、综合结论

| 维度 | 结论 |
|------|------|
| **样本外泛化（OOS 2016-2017）** | ✅ OOS Alpha +14~16%，策略有泛化能力 |
| **tstat（OOS）** | ⚠️ 边缘显著（1.59~1.96），Conservative 接近95%置信 |
| **主回测（2018-2025）** | ✅ Aggressive +281% / Conservative +252% 均跑赢 SPY +188% |
| **tstat（2018-2025）** | ❌ 弱显著（0.20~0.73），Alpha 绝对值有限，含2022熊市噪音大 |
| **tstat（2025-2026）** | ❌ 负值（-1.07~-0.57），近期跑输 SPY |
| **下行风险控制** | ✅ 三个回测区间策略最大回撤均优于 SPY |
| **过拟合风险** | 低（OOS 样本外仍有正 Alpha；2018-2025 历史财报模式亦有正 Alpha）|

> **后续对比说明**：切换数据源（FMP）后，用同样的脚本跑一次相同时间段回测，
> 对比 tstat、Sharpe、Alpha、MaxDD 等指标是否有明显变化。
> 若指标基本一致，说明数据源切换没有引入偏差；若差异显著，需排查 FMP 数据质量。
