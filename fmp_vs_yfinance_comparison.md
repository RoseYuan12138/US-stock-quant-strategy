## 重要数据源差异说明

在解读指标差异之前，需要了解两个数据源的根本性差异：

| 维度 | yfinance | FMP parquet |
|------|----------|-------------|
| **价格类型** | 复权收盘价（split + 股息再投资调整） | 仅拆股复权（无股息调整） |
| **股息处理** | 已折算回历史价格中 | 不包含，只有原始收盘价 |
| **对绝对收益的影响** | 含股息，收益偏高 | 不含股息，收益偏低 |
| **对 Alpha 的影响** | 组合与基准都含股息，相互抵消 | 组合与基准都不含股息，相互抵消 |
| **实际差距（8年SPY）** | +187.5%（含股息再投资） | +153.7%（纯价格涨幅） |

> **结论**：FMP 的绝对收益数字会系统性低于 yfinance ~25-35pp（因缺少股息），但**Alpha（超额收益）在两个数据源内部是自洽的**。直接比较绝对收益无意义，应重点对比 Alpha 方向和 Sharpe。

### 3.3 V3 主回测差异分析

Aggressive 配置差异最大（yfinance +281% vs FMP +213%），主要原因分析：

| 原因 | 说明 |
|------|------|
| **股息调整差异（最主要）** | yfinance 含股息再投资，8年累积约+15~20%。Aggressive 持仓多、换手高，股息影响被放大。 |
| **AI 牛市期选股差异** | 2023-2024 年 NVDA/MSFT 等高动量股票的价格时序可能有差异，影响选入时机 |
| **动量因子数值差异** | 价格历史不同 → 动量评分不同 → 选股略有差异 → 长期累积 |

## 四、综合差异分析

### 4.1 Alpha 方向一致性

| 回测区间 | yfinance Alpha 方向 | FMP Alpha 方向 | 是否一致 |
|--------|-------------------|---------------|--------|
| OOS 2016-2017 | ✅ 正（+13.9%~+16.2%） | ✅ 正（+15.5%~+19.4%） | **一致** |
| 主回测 2018-2025 | ✅ 正（+0.1%~+4.1%） | ✅ 正（+2.7%~+3.2%） | **一致** |
| 近期 2025-2026 | ❌ 负（-14.0%~-7.7%） | ❌ 负（-12.4%~-4.0%） | **一致** |

**Alpha 方向在所有三个时间段均与 yfinance 版本完全一致**，说明策略逻辑是稳健的。

### 4.2 核心指标变化幅度

| 指标类别 | 变化幅度 | 原因 |
|---------|---------|------|
| 绝对收益（总/年化） | 变化较大（因股息差） | FMP 无股息调整，系统性偏低 |
| **Alpha** | 变化适中（±1~5pp） | 数据源内自洽，FMP OOS alpha偏高 |
| **Sharpe** | 变化较小（±0.05~0.32） | 主要受 alpha 差异影响 |
| **最大回撤** | 几乎不变（±0~2pp） | 价格路径相似 |
| **胜率** | 变化较大（+2~7pp） | FMP 价格数据有差异，影响交易盈亏判断 |
| Earnings 因子 | FMP 数据更完整（79条 vs yfinance可变） | FMP earnings 覆盖更长历史 |

### 4.3 为什么 FMP OOS Alpha 更高？

OOS 期间（2016-2017）FMP Alpha 比 yfinance 高约 +2~5pp，可能原因：
1. yfinance 2015-2017 的历史数据可能有缺失或调整差异
2. FMP 作为专业金融数据源，这期间的价格更准确
3. 更准确的价格 → 动量因子更准 → 选股质量更高

## 五、数据源切换结论

| 维度 | 结论 |
|------|------|
| **代码切换成功** | ✅ FMP parquet 被正确读取，adaptor 接口零改动 |
| **策略逻辑不变** | ✅ strategy/ 和 backtest/ 层零改动验证通过 |
| **Alpha 方向一致** | ✅ 三段回测 Alpha 正/负方向 100% 与 yfinance 一致 |
| **数据质量差异** | ⚠️ FMP = 拆股复权但无股息调整；yfinance = 全总收益调整 |
| **绝对收益偏低** | ⚠️ FMP 系统性低约 25-35pp（长期），属正常数据特征差异 |
| **Alpha 量级差异** | ⚠️ OOS 期间 FMP alpha 明显更高（19.4% vs 13.9%），需关注 |
| **切换是否成功** | ✅ **成功** — 策略有效性不变，只有价格处理方式不同 |

### 建议

1. **如需对齐 yfinance 结果**：使用 FMP 的 `vwap` 或从 FMP API 获取调整后价格（`historical-price-full` with `adjusted=true`）
2. **如需纯价格回测**：FMP 当前数据是准确的，Alpha 方向可信
3. **Earnings 数据**：FMP earnings parquet 覆盖更完整（AAPL 79条记录），切换后 PEAD 因子数据质量提升
4. **后续 fallback 机制**：已实现 FMP → yfinance fallback，股票不在 FMP 库中时自动回退
